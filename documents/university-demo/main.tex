\section{Features}

\subsection{Mathematics Support}

The template fully supports \LaTeX{} mathematical expressions. Inline mathematics works as expected: $E = mc^2$. Display mathematics:

\begin{equation}
    \int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
\end{equation}

\subsection{Code Listings}

Multiple programming languages are supported:

\begin{lstlisting}[style=pythonstyle, caption=Python Example]
def calculate_average(numbers: list) -> float:
    """Calculate the average of a list of numbers."""
    return sum(numbers) / len(numbers)

data = [1, 2, 3, 4, 5]
avg = calculate_average(data)
print(f"Average: {avg}")
\end{lstlisting}
