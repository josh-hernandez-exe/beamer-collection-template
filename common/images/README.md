# Shared Images

Place reusable images and diagrams here (PNG, PDF, SVG).

These are accessible from all presentations and documents via the `graphicspath` configuration that includes `common/images/`.

Example use in a presentation or document:

```latex
\includegraphics[width=0.5\textwidth]{my-shared-image.pdf}
```

Supported formats:
- PDF (vector, recommended)
- SVG (vector, converted to PDF for LaTeX)
- PNG (raster, acceptable for photographs)
- JPG (raster, acceptable for photographs)
